											Program Controls
************************************************************************************************************
Move Link: W(Up), A(Left), S(Down), D(Right)
Damage Link: E
Link Items: 1(Green Arrow), 2(Blue Arrow), 3(Normal Boomerang), 4(Blue Boomerang), 5(Bomb), 6(Fireball)
Attack: Z, N, Spacebar
Quit: Q, Escape
Restart: R
Changing Room: Left and right mouse buttons
